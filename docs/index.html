Ketch

Ketch is a really simple way to start new projects.

Pick a template, answer a few questions, and your project is setup.

How It Works

A ketch template is just a github repo of text files using mustache syntax. Here's a simple PHP project template, with a composer.json and README.md:


{
    "name": "{{ vendor }}/{{ package }}",
    "type": "library",
    "description": "{{ description }}",
    "license": "MIT",
    "require": {
        "php" : "~5.5|~7.0"
    },
    "require-dev": {
        "phpunit/phpunit" : "4.*"
    },
    "autoload": {
        "psr-4": {
            "{{ vendor|case.pascal }}\\{{ package|case.pascal }}\\": "src"
        }
    }
}
        

# {{ package|case.upper }}

{{ description }}

## Install

    $ composer require {{ vendor }}/{{ package }}

Ketch will ask you to fill in the blanks:


$ ketch create yuloh/skeleton container

vendor: yuloh
package: container
description: An excellent dependency injection container.

Project ready!

...and your new project is created.
